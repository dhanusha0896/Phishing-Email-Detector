# Phishing-Email-Detector
The Phishing Email Detector is a web-based tool that analyzes email content to identify potential phishing indicators using pattern recognition and contextual heuristics. It highlights suspicious elements such as urgency-based language, mismatched sender domains, suspicious links, and insecure URLs .
